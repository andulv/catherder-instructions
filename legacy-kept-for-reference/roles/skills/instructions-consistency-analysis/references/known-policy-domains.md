# Known Policy Domains

Use this list to drive consistency comparisons across `.instructions/`.
Expand as the instruction set evolves.

## Initial Domains

- Timestamp format and update conventions
- Task execution workflow (inline vs separate tasks, commit rules)
- File naming conventions
- Front-matter schema requirements
- Reading order and entry-point behavior
- Role assignment boundaries and ownership

## Classification Guidance

For each domain comparison, classify findings as:
- `CONFLICT` — incompatible guidance
- `DUPLICATE` — same rule repeated without canonical reference
- `AMBIGUOUS` — overlap without explicit precedence
- `CONSISTENT` — aligned guidance
